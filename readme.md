data      - 测试用例文件
log       - 日志
base      - 基础文件，封装请求方法
report    - 可视化测试报告
utils     - 工具
main      - 脚本运行主目录
testcase  - 测试脚本
README.md - 框架说明

2.获取了环境依赖包后，在新环境下安装依赖包的模块：pip install ./requirements.txt
